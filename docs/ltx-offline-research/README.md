# LTX Offline Fork Research

This directory contains the research used to plan the local-model-friendly LTX-Desktop fork. It was copied into the repo so Pi agents on another machine have the same context after `git clone`.

Read order:

1. `01-ltx-2-architecture-and-official-stack.md` — upstream LTX-2 architecture and official stack.
2. `02-quantization-and-low-vram-options.md` — low-VRAM, quantization, GGUF options.
3. `03-standalone-app-architecture-options.md` — standalone app options and tradeoffs.
4. `04-recommended-implementation-plan.md` — initial recommended plan.
5. `05-ltx-desktop-model-profiles-and-gguf-kijai-plan.md` — model profile and GGUF/Kijai design.
6. `06-revised-ltx-desktop-implementation-roadmap.md` — revised roadmap after LTX-Desktop review.
7. `07-official-ltx23-lora-registry.md` — official adapter/LoRA registry source for next milestone.

Current stage: Milestone 1 complete. Next: official LTX-2.3 adapter registry and backend status/recommendation endpoints.
